# Rime五笔86版配置方案

## 特性
- 五笔拼音混输
- 智能造词，用户词序调频，系统词序固定，系统词优先于用户词
- 按```;```进入临时英文模式
- 按```~```进入以形查音模式
- 按```/```进行特殊符号快速输入
- 分号、单引号上屏二三候选词

## 配置修改方法

将```wubi_smart.custom.yaml```中相关功能的代码注释/反注释

QQ五笔系统词库
